# Arclink

Send and receive USDC using only an email address.

Arclink is a modern payment application built on Arc and Circle infrastructure that enables users to send and receive USDC without needing to understand wallet addresses, blockchain networks, or gas fees.

## Features

* Email-based USDC transfers
* Dynamic authentication
* Circle Wallet integration
* USDC payments
* Transaction history
* User dashboard
* Secure identity mapping
* Mobile-friendly experience

## Tech Stack

* Next.js
* TypeScript
* Dynamic
* Circle Developer Platform
* USDC
* Supabase
* Vercel

## How It Works

1. User signs up with email.
2. A wallet is created and linked automatically.
3. Users send USDC using an email address.
4. Recipients receive funds without needing to share wallet addresses.
5. Transactions are recorded and displayed in the dashboard.

## Vision

Arclink makes digital dollar payments as simple as sending an email.
